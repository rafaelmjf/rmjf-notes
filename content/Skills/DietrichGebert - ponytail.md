A **"lazy senior developer" skill** for coding agents (Claude Code, Codex, Copilot CLI) built on the principle that "the best code is the code you never wrote." Before writing code it applies a decision ladder: is it necessary → in the stdlib → a native platform feature → an installed dependency → a one-liner — and only then writes new code.

**Commands:** `/ponytail-review` (diff analysis), `/ponytail-audit` (repo-wide), `/ponytail-debt` (deferred-optimization tracking), `/ponytail-gain` (impact metrics). Reported benchmarks: ~54% less code, ~20% lower cost, ~27% faster.
